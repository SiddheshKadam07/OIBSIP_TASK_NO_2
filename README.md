# ⚖️ BMI Calculator

## 📌 Project Description

The BMI Calculator is a Python application that calculates a person's Body Mass Index (BMI) using their height and weight. It also shows the BMI category and stores records in a database.

This project was developed as part of the **Oasis Infobyte Internship (OIBSIP)**.

---

## ✨ Features

* Calculate BMI
* Show BMI category
* Save BMI records
* View BMI history
* Support Metric and Imperial units
* Bulk BMI calculation
* Simple and easy-to-use interface

---

## 🛠️ Technologies Used

* Python
* Tkinter
* SQLite

---

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```

---
## 📊 BMI Categories

| BMI Range | Category |
|-----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and Above | Obese |
